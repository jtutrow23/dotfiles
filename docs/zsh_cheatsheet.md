# 🐚 Zsh Cheat Sheet

## Environment
- Manager: Znap
- Prompt: Starship
- Modular files:
  - options.zsh
  - aliases.zsh
  - functions.zsh
  - extras.zsh

## Commands
- warm → update + precompile Zsh configs
- brew bundle → install from Brewfile
- macos_bootstrap.sh → apply defaults

## Shortcuts
- gs → git status
- ga → git add
- mkcd dir → mkdir + cd into dir
- extract file.tar.gz → extract common archive formats
- updateall → brew + plugins update
